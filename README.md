# NBB Health Dashboard

Private pregnancy health tracking dashboard for the Bhagat family.

## Pages
- **Blood Work Dashboard** (`index.html`) — Interactive charts tracking CBC, glucose, lipids, thyroid, liver & kidney function
- **Pregnancy Tracker** (`pregnancy.html`) — Timeline, prescriptions, risk factors, upcoming milestones
- **Ultrasound Gallery** (`ultrasound.html`) — Scan history, growth charts, anatomy findings
- **Diet & Nutrition** (`nutrition.html`) — Wheat-free, milk-free meal plans, supplement schedule, iron-boosting guide

## How to Update
1. Add new PDF reports to the data source folder
2. Update `data/bloodwork.json` with new values
3. Push changes — GitHub Pages auto-deploys

## Tech
- Static HTML/CSS/JS — no build step needed
- Chart.js for interactive charts
- Soft pastel Material Design theme
- Mobile responsive
